Venture Funding Monitor

Automated funding intelligence for startup discovery

Challenge / Context

Identifying relevant funding opportunities across UK, European, and Middle Eastern markets required manually monitoring dozens of news sources daily. This time-intensive process risked missing time-sensitive opportunities and lacked systematic tracking of emerging companies in target sectors (fintech, SaaS).

Approach & Technologies

Built an automated RSS monitoring system using Python that processes 100-200 articles daily from 9 curated sources. Implemented regex-based pattern matching to extract funding details (stage, amount, location, industry) and a scoring algorithm that prioritizes opportunities based on geographic and sector relevance. Deployed on GitHub Actions for zero-cost cloud automation with SQLite for deduplication and historical tracking.

Python SQLite GitHub Actions Data Pipelines Automation

Key Insights / Outcomes

  • Automated discovery of 5-10 qualified funding announcements daily, reducing manual research time from 2+ hours to zero
  • Relevance scoring system ensures 80%+ precision, filtering noise while capturing UK-focused Series A-C opportunities
  • Historical database enables trend analysis: identifying hot sectors, funding velocity, and geographic shifts in venture capital
  • Demonstrated product thinking: prioritized simplicity (regex vs. LLM) to minimize costs while maintaining effectiveness

FTSE Trading Strategy Backtester

Quantitative analysis framework for mean-reversion strategies

Challenge / Context

Validating trading strategies requires rigorous statistical testing against historical data, accounting for realistic market conditions (commissions, slippage, drawdowns). Manual backtesting is error-prone and lacks reproducibility. The goal was to build a systematic framework to evaluate mean-reversion strategies on FTSE 100 opening price deviations.

Approach & Technologies

Developed a modular Python backtesting engine with three core layers: strategy logic (signal generation based on price deviations), portfolio management (position tracking with realistic cost modeling), and performance analytics (risk-adjusted metrics). Implemented object-oriented design patterns for extensibility, allowing rapid testing of parameter variations and new strategy rules.

Python Pandas NumPy Statistical Analysis Financial Modeling

Key Insights / Outcomes

  • Comprehensive risk metrics (Sharpe ratio, max drawdown, win rate) revealed strategy viability: 62% win rate with 1.76 profit factor on sample data
  • Identified optimal entry thresholds: ±10 point deviations balanced trade frequency with risk-adjusted returns
  • Drawdown analysis highlighted vulnerability during high-volatility periods, informing risk management rules (max deviation filters)
  • Framework enables data-driven iteration: test hypothesis → analyze metrics → refine strategy, mirroring product experimentation cycles
